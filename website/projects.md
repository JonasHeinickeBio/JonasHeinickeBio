### Ontology Mapper
**Technologies**: Python, RDF, BioPortal API, OLS API  
**Description**: A modular CLI and GUI tool for mapping and enriching ontologies using BioPortal and OLS APIs. Supports 24+ biomedical ontologies including MONDO, HP, NCIT, DOID, CHEBI, and more.

**Key Features**:
- Multi-ontology support (MONDO, HP, NCIT, DOID, CHEBI, GO, SNOMEDCT, etc.)
- Dual API integration with intelligent fallback
- Multiple input/output formats (Turtle, JSON-LD, RDF/XML, SSSOM)
- Interactive CLI and user-friendly GUI
- Intelligent caching for performance
- Batch processing capabilities

**Use Cases**: Disease research, symptom analysis, chemical/drug research, ontology enrichment

[View on GitHub](https://github.com/JonasHeinickeBio/ontology-mapper)
